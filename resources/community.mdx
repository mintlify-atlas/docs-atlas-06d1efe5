## Community Guidelines

### How to Get Help

<Steps>
  <Step title="Search First">
    Check existing GitHub issues and discussions for similar questions
  </Step>
  
  <Step title="Ask on Discord">
    For quick questions, join our Discord and ask in the appropriate channel
  </Step>
  
  <Step title="Create an Issue">
    For bugs or feature requests, open a detailed GitHub issue
  </Step>
  
  <Step title="Share Your Success">
    When you solve a problem, share your solution to help others
  </Step>
</Steps>

### Best Practices

<AccordionGroup>
  <Accordion title="Asking Questions">
    - Provide context about what you're trying to achieve
    - Include error messages and logs
    - Mention your environment (OS, Python/Node versions)
    - Show what you've already tried
  </Accordion>
  
  <Accordion title="Reporting Bugs">
    - Use a clear, descriptive title
    - Provide steps to reproduce
    - Include expected vs actual behavior
    - Add screenshots if relevant
    - Specify your MiroFish version
  </Accordion>
  
  <Accordion title="Sharing Ideas">
    - Search for similar feature requests first
    - Explain the use case and benefits
    - Be open to alternative solutions
    - Consider implementation complexity
  </Accordion>
</AccordionGroup>
