# Introduction to MiroFish

**MiroFish** is a next-generation AI prediction engine powered by multi-agent technology. By extracting seed information from the real world (such as breaking news, policy drafts, or financial signals), it automatically constructs a high-fidelity parallel digital world. Within this space, thousands of intelligent agents with independent personalities, long-term memory, and behavioral logic freely interact and undergo social evolution.

## What Makes MiroFish Special

You can inject variables dynamically from a "God's-eye view" to precisely deduce future trajectories — **rehearse the future in a digital sandbox, and win decisions after countless simulations**.

<Note>
**Simple Input, Powerful Output**

You only need to:
- Upload seed materials (data analysis reports or interesting novel stories)
- Describe your prediction requirements in natural language

MiroFish will return:
- A detailed prediction report
- A deeply interactive high-fidelity digital world
</Note>

## Key Capabilities

### Multi-Agent Swarm Intelligence

Thousands of intelligent agents with:
- **Independent personalities** - Each agent has unique traits and behaviors
- **Long-term memory** - Agents remember past interactions and events
- **Behavioral logic** - Sophisticated decision-making processes
- **Social evolution** - Emergent behaviors from agent interactions

### High-Fidelity Digital Worlds

MiroFish constructs parallel digital environments by:
- Extracting seed information from real-world data
- Building knowledge graphs with GraphRAG technology
- Generating realistic social networks and relationships
- Simulating authentic human interactions

### Dynamic Prediction Engine

Powerful prediction capabilities:
- **Variable injection** - Introduce new factors during simulation
- **Temporal evolution** - Track how situations develop over time
- **Multiple scenarios** - Test different "what-if" conditions
- **Detailed reporting** - Comprehensive analysis with interactive tools

## Use Cases

MiroFish has been successfully applied to:

- **Public Opinion Analysis** - Predict how events will unfold in social media
- **Policy Impact Assessment** - Test policy changes in a risk-free environment
- **Creative Storytelling** - Explore alternative endings for novels and narratives
- **Financial Forecasting** - Model market reactions to various scenarios
- **Crisis Management** - Simulate response strategies before implementation

<Warning>
**Resource Consumption Notice**

MiroFish simulations consume significant LLM API resources. We recommend starting with simulations of fewer than 40 rounds to understand resource usage before running larger experiments.
</Warning>

## Technology Stack

MiroFish is built on cutting-edge technologies:

- **Simulation Engine**: Powered by [OASIS (Open Agent Social Interaction Simulations)](https://github.com/camel-ai/oasis) from CAMEL-AI
- **Memory Management**: Zep Cloud for scalable agent memory and knowledge graphs
- **LLM Integration**: OpenAI-compatible API format supporting multiple providers
- **Frontend**: Vue 3 with interactive visualization
- **Backend**: Flask with Python 3.11+
