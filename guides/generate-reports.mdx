## Overview

After running a simulation, MiroFish's Report Agent can generate detailed analytical reports. The agent:

- **Autonomous Analysis**: Explores simulation data using specialized tools
- **Multi-Section Reports**: Executive summary, findings, sentiment analysis, predictions
- **GraphRAG Integration**: Queries knowledge graph for context and insights
- **Interactive Chat**: Continue analysis through conversational queries

<Note>
  Report generation is an asynchronous LLM-intensive process that typically takes 5-15 minutes depending on simulation size.
</Note>

<Steps>
  <Step title="Initiate report generation">
    Start the Report Agent to analyze your completed simulation:

    <CodeGroup>
      ```bash cURL
      curl -X POST http://localhost:5001/api/report/generate \
        -H "Content-Type: application/json" \
        -d '{
          "simulation_id": "sim_xyz789"
        }'
      ```

      ```python Python
      import requests

      response = requests.post(
          "http://localhost:5001/api/report/generate",
          json={"simulation_id": "sim_xyz789"}
      )

      result = response.json()
      report_id = result['data']['report_id']
      task_id = result['data']['task_id']
      
      print(f"Report ID: {report_id}")
      print(f"Task ID: {task_id}")
      ```

      ```javascript JavaScript
      const response = await fetch('http://localhost:5001/api/report/generate', {
        method: 'POST',
        headers: {'Content-Type': 'application/json'},
        body: JSON.stringify({simulation_id: 'sim_xyz789'})
      });

      const {data} = await response.json();
      console.log('Report ID:', data.report_id);
      console.log('Task ID:', data.task_id);
      ```
    </CodeGroup>

    Response includes both report_id and task_id:

    ```json
    {
      "success": true,
      "data": {
        "simulation_id": "sim_xyz789",
        "report_id": "report_abc456",
        "task_id": "task_rep_789",
        "status": "generating",
        "message": "报告生成任务已启动，请通过 /api/report/generate/status 查询进度",
        "already_generated": false
      }
    }
    ```
  </Step>

  <Step title="Monitor generation progress">
    Track report generation across multiple sections:

    <CodeGroup>
      ```bash cURL
      curl -X POST http://localhost:5001/api/report/generate/status \
        -H "Content-Type: application/json" \
        -d '{"task_id": "task_rep_789"}'
      ```

      ```python Python
      import requests
      import time

      task_id = "task_rep_789"
      url = "http://localhost:5001/api/report/generate/status"

      while True:
          response = requests.post(url, json={"task_id": task_id})
          task = response.json()['data']
          
          print(f"{task['status']}: {task['message']} ({task['progress']}%)")
          
          if task['status'] in ['completed', 'failed']:
              print(f"Final result: {task.get('result')}")
              break
          
          time.sleep(5)
      ```

      ```javascript JavaScript
      async function monitorReportGeneration(taskId) {
        while (true) {
          const response = await fetch(
            'http://localhost:5001/api/report/generate/status',
            {
              method: 'POST',
              headers: {'Content-Type': 'application/json'},
              body: JSON.stringify({task_id: taskId})
            }
          );
          
          const {data: task} = await response.json();
          console.log(`${task.status}: ${task.message} (${task.progress}%)`);
          
          if (['completed', 'failed'].includes(task.status)) {
            return task;
          }
          
          await new Promise(r => setTimeout(r, 5000));
        }
      }
      ```
    </CodeGroup>

## Interactive Report Chat

After report generation, continue analysis through conversation:

<CodeGroup>
  ```bash cURL
  curl -X POST http://localhost:5001/api/report/chat \
    -H "Content-Type: application/json" \
    -d '{
      "simulation_id": "sim_xyz789",
      "message": "What were the most controversial topics?",
      "chat_history": []
    }'
  ```

  ```python Python
  import requests

  response = requests.post(
      "http://localhost:5001/api/report/chat",
      json={
          "simulation_id": "sim_xyz789",
          "message": "What were the most controversial topics?",
          "chat_history": []
      }
  )

  result = response.json()
  print("Agent:", result['data']['response'])
  print("Tools used:", result['data']['tool_calls'])
  ```

  ```javascript JavaScript
  const response = await fetch('http://localhost:5001/api/report/chat', {
    method: 'POST',
    headers: {'Content-Type': 'application/json'},
    body: JSON.stringify({
      simulation_id: 'sim_xyz789',
      message: 'What were the most controversial topics?',
      chat_history: []
    })
  });

  const {data} = await response.json();
  console.log('Response:', data.response);
  ```
</CodeGroup>

## Report Structure

Typical report includes:

1. **Executive Summary**: High-level insights and key takeaways
2. **Simulation Context**: Background, parameters, and methodology
3. **Key Findings**: Major discoveries and patterns
4. **Sentiment Analysis**: Opinion trends and evolution over time
5. **Platform Comparison**: Reddit vs Twitter dynamics
6. **Influential Agents**: Key opinion leaders and their impact
7. **Predictions & Recommendations**: Future implications and suggested actions
