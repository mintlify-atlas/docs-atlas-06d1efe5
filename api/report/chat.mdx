## POST /api/report/chat

Engage in interactive conversation with the Report Agent to ask questions about simulation results, request additional analysis, or get clarifications on report content. The Agent autonomously uses retrieval tools to search knowledge graphs and provide contextual answers.

## How It Works

The Report Agent:

1. Receives your question along with conversation history
2. Analyzes the query and determines which tools to use
3. Searches the knowledge graph for relevant information
4. Retrieves simulation events, entities, and relationships
5. Generates a comprehensive answer with citations
6. Returns the response along with tool call details and sources

<Info>
The Agent maintains context from previous messages in the conversation, allowing for follow-up questions and clarifications.
</Info>

## Request

<ParamField body="simulation_id" type="string" required>
  The simulation ID to query about (format: `sim_xxxx`)
</ParamField>

<ParamField body="message" type="string" required>
  The user's question or message to the Report Agent
</ParamField>

<ParamField body="chat_history" type="array">
  Optional conversation history for context. Array of message objects with `role` and `content`.
  
  ```json
  [
    {"role": "user", "content": "What are the key findings?"},
    {"role": "assistant", "content": "The main findings are..."}
  ]
  ```
</ParamField>

```python Python
import requests

history = []

def chat(message):
    response = requests.post(
        "http://localhost:5000/api/report/chat",
        json={
            "simulation_id": "sim_a1b2c3d4e5f6",
            "message": message,
            "chat_history": history
        }
    )
    
    result = response.json()["data"]
    
    # Update history
    history.append({"role": "user", "content": message})
    history.append({"role": "assistant", "content": result["response"]})
    
    return result

# Multi-turn conversation
result1 = chat("What are the main findings?")
print(result1["response"])

result2 = chat("Can you elaborate on the first finding?")
print(result2["response"])
```

```javascript JavaScript
let chatHistory = [];

async function chat(message) {
  const response = await fetch('http://localhost:5000/api/report/chat', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({
      simulation_id: 'sim_a1b2c3d4e5f6',
      message: message,
      chat_history: chatHistory
    })
  });
  
  const { data } = await response.json();
  
  // Update history
  chatHistory.push({ role: 'user', content: message });
  chatHistory.push({ role: 'assistant', content: data.response });
  
  return data;
}

// Example usage
const result = await chat('请解释一下舆情走向');
console.log(result.response);
console.log('Tools used:', result.tool_calls);
```

## Response

<ResponseField name="success" type="boolean">
  Indicates whether the chat request was successful
</ResponseField>

<ResponseField name="data" type="object">
  <ResponseField name="response" type="string">
    The Agent's text response to the user's question
  </ResponseField>
  
  <ResponseField name="tool_calls" type="array">
    List of tools the Agent invoked to answer the question. Each tool call includes:
    - `tool_name`: Name of the tool (e.g., `search_graph`, `insight_forge`)
    - `parameters`: Parameters passed to the tool
    - `result`: Tool execution result
  </ResponseField>
  
  <ResponseField name="sources" type="array">
    Information sources used to generate the response, including:
    - Knowledge graph entities
    - Simulation events
    - Timeline data
    - Citations and references
  </ResponseField>
</ResponseField>

## Best Practices

<AccordionGroup>
  <Accordion title="Maintain Conversation Context">
    Always include `chat_history` for multi-turn conversations. This allows the Agent to understand follow-up questions and references to previous answers.
  </Accordion>
  
  <Accordion title="Ask Specific Questions">
    More specific questions yield better results. Instead of "Tell me about the simulation", ask "What were the sentiment changes during the T3-T5 period?"
  </Accordion>
  
  <Accordion title="Reference Previous Responses">
    You can ask the Agent to elaborate on specific points from previous answers: "You mentioned X earlier, can you explain that in more detail?"
  </Accordion>
  
  <Accordion title="Check Tool Calls">
    Review the `tool_calls` field to understand what data the Agent retrieved and how it formulated the answer.
  </Accordion>
  
  <Accordion title="Verify Sources">
    The `sources` field provides transparency into which entities, events, and facts contributed to the response.
  </Accordion>
</AccordionGroup>

## Integration Example

<CodeGroup>
```python Full Chatbot
import requests
from typing import List, Dict

class ReportChatbot:
    def __init__(self, simulation_id: str, base_url: str = "http://localhost:5000"):
        self.simulation_id = simulation_id
        self.base_url = base_url
        self.history: List[Dict[str, str]] = []
    
    def ask(self, message: str) -> Dict:
        """Send a message to the Report Agent"""
        response = requests.post(
            f"{self.base_url}/api/report/chat",
            json={
                "simulation_id": self.simulation_id,
                "message": message,
                "chat_history": self.history
            }
        )
        
        if not response.ok:
            raise Exception(f"API error: {response.json()['error']}")
        
        result = response.json()["data"]
        
        # Update history
        self.history.append({"role": "user", "content": message})
        self.history.append({"role": "assistant", "content": result["response"]})
        
        return result
    
    def reset(self):
        """Clear conversation history"""
        self.history = []

# Usage
bot = ReportChatbot("sim_a1b2c3d4e5f6")

# First question
result = bot.ask("What are the main findings?")
print("Agent:", result["response"])
print("Sources:", len(result["sources"]), "sources used")

# Follow-up question (context maintained)
result = bot.ask("Can you elaborate on the first finding?")
print("Agent:", result["response"])
```
